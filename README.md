# Cradit-caThis project focuses on detecting fraudulent credit card transactions using a hybrid anomaly detection approach. It combines both supervised and unsupervised machine learning techniques to improve the accuracy of fraud detection, especially in highly imbalanced datasets. The workflow includes data preprocessing, exploratory data analysis (EDA), and implementation of models such as Logistic Regression and Decision Tree for supervised learning, along with K-Means and DBSCAN for unsupervised anomaly detection. The models are evaluated using performance metrics like precision, recall, and F1-score. By integrating multiple approaches, the system enhances the ability to identify rare fraudulent activities more effectively compared to using a single model. The project is implemented using Python with libraries such as Scikit-learn, Pandas, NumPy, and Matplotlib.rd-fraud-detection-hybride-anomaly
